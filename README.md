# 🚀 React Step Counter App

> Một ứng dụng đơn giản giúp bạn tăng/giảm ngày dựa trên số bước (`step`) và giá trị (`count`).  
> Được xây dựng với **React** và **CSS Flexbox** để có giao diện gọn gàng, đẹp mắt.
